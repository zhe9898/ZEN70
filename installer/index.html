Cloudflare 终极图文抢险教程 (解决 1033 报错)

极其重要

1. 第一步：只拿密码，千万不要照着图去装软件！

在这个只属于 Cloudflare 官方的黑框页面里，它会让你下载 .msi、运行命令行... 如果这样做了，你就上大当了！不要下载那个东西！

你真正要做的：在最下方那段又臭又长的长代码里，用鼠标选中最末尾 eyJh 开头的一串超长火星文，右键复制（不要复制前面的英文命令！）。然后把它粘贴到咱们这个 ZEN70 部署网页的「外网穿透令牌」框里。

... cloudflared.exe service install eyJh...在这复制超长乱码...
复制完密码后不要关闭，点击网页右下角蓝色的 【Next (下一步)】 或 【Save】！！

2. 第二步：绑定门牌号（解决 1033 的核心）

绝大部分人在第一步复制完代码就把 Cloudflare 关了，这是致命的大错！你刚才拿了令牌，证明门卫认识这台电脑了，但还要给门卫指出你的「房间号」在哪，否则你访问外网域名就是白屏 Error 1033！

  • 点击 Next 之后，照这样填写「添加路线 (Public Hostname)」：
    上层：外网身份
    Subdomain: 留空不填
    Domain: 选你的 zen70.cn
    Path: 留空不填
    下层：家里暗号 (必须原样填!)
    服务网址: 直接手动填入 http://localhost:8000
    Cloudflare 新版界面把协议和地址合并了，直接把这串完整带 http 的网址打进去即可，千万别写成 https！

    填完后点右下角蓝色的 [添加路线 (Save hostname)] 按钮即可！大门彻底打通！

🚀 ZEN70 部署避坑与下沉扩展完全指南

🛑 避坑 1：刚点完部署 CPU 怎么疯了？电脑会卡死吗？

真相：这其实是你的杀毒软件（比如 Windows Defender 的 MsMpEng.exe）在作祟！系统拉起海量架构文件时，杀毒软件这种“死心眼”会疯狂扫描每一个针眼大的读写动作，活活累死 CPU。

🌟 终极解决方案（极大提升部署速度）：
  • 按 Win键 搜索并打开 Windows 安全中心。
  • 进入 病毒和威胁防护 -> 点击蓝色的 管理设置。
  • 滚动到底部 排除项 -> 点击 添加或删除排除项。
  • 点击 [+添加排除项] -> 选 [文件夹]，把你放刚下载的这些代码的那个文件夹选进去！

加入白名单后，CPU 立刻释放性能恢复安静，系统部署直接起飞！

🛑 避坑 3：报错 1033！外网访问失败？

如果你的隧道连接成功（Terminal 不报错），但手机用流量访问却看到 Cloudflare 发来的白屏 Error 1033 错误。

那是因为门卫还不认识你的门牌号！请退出去，点击【外网穿透令牌】密码框旁边的蓝色小按钮 "不会填？看图文教程" 照做最后一步（Service 填 HTTP，URL 填 localhost:8000）。
